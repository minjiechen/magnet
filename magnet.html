magnet.core.core_loss_iGSE_arbitrary(freq, flux, duty, k_i=None, alpha=None, beta=None, material=None, n_interval=10000)¶

Calculate magnetic core loss using iGSE without minor loops

Parameters

  • freq – Frequency of excitation waveform (Hz)

  • flux – Relative Flux Density (T) in a single waveform cycle, as an ndarray

  • duty – Fractional time wrt time period, in [0, 1], in a single waveform cycle, as an ndarray

  • k_i – Steinmetz coefficient k_i

  • alpha – Steinmetz coefficient alpha

  • beta – Steinmetz coefficient beta

  • material – Name of material. If specified, k_i/alpha/beta are ignored.

  • n_interval – No. of intervals to use to solve iGSE using trapezoidal rule

Returns

Core loss (W/m^3)
